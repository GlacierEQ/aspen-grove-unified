# Aspen Grove Pillar Profile

## Repository
`GlacierEQ/aspen-grove-unified`

## Role
Aggregate/control-plane presentation surface.

## Mission
Aggregate Aspen Grove doctrine and runtime surfaces into a readable, navigable control-plane view.

## Root authority
This pillar inherits root doctrine from:

```text
GlacierEQ/aspen-grove-operator-v7/APEX_CENTER/
```

It is not a separate root authority.

## Primary piston inputs
- root doctrine updates
- registry snapshots
- status summaries

## Primary piston outputs
- aggregated indices
- control-plane dashboards
- family visibility artifacts

## Default command center
`publisher_command_center`

## Allowed connectors
- GitHub
- Notion
- PDF/doc generation

## Forbidden behavior
- redefining root doctrine
- acting as root authority
- storing raw secrets
- publishing unverified material as final

## Checkpoint rule
Every bounded deployment or derived artifact should checkpoint back to Aspen Grove Center.
