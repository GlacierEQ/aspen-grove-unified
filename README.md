## 🚀 Quick Start

### Prerequisites
- Python 3.10+
- Node.js 18+
- Docker & Docker Compose
- Kubernetes (optional, for production)

### Local Development

```bash
# Clone & setup
git clone https://github.com/GlacierEQ/aspen-grove-unified.git
cd aspen-grove-unified

# Python environment
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Node.js
npm install

# Initialize quantum systems
npm run quantum-init
npm run memory-sync

# Start dev server
npm run dev
# Python backend: python core/server.py
```

### Docker Deployment

```bash
# Build unified image
npm run docker:build

# Run full stack
docker-compose up -d

# Verify
docker-compose ps
```

### Kubernetes Deployment

```bash
# Deploy all services
npm run k8s:deploy

# Monitor
kubectl get pods
kubectl logs -f deployment/aspen-grove-unified
```
